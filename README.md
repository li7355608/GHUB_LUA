# GHUB_LUA

基于 **Logitech G HUB** 的 Lua 鼠标宏脚本集合，提供辅助操作脚本。

## 项目结构

```
game/
├── apex/           # Apex Legends
│   └── R301.lua    # R-301 压枪
├── strinova/       # 卡拉彼丘 (Strinova)
│   ├── NormalGun.lua        # 通用压枪
│   ├── AutomaticMachine.lua # 极限射速连点
│   └── KanamiMadeleine.lua  # 瞬狙弦化
```

## 使用方式

1. 打开 **Logitech G HUB**
2. 进入设备的 **脚本编辑器**
3. 将对应脚本的内容粘贴到编辑器中
4. 使用鼠标 **G8 侧键** 切换脚本开关，CapsLock 灯指示当前状态
